# TrademarkCheck 🔍

**Free Public Trademark Search India**

[![Live Site](https://img.shields.io/badge/Live%20Demo-trademarkcheck--sg.vercel.app-blue?style=for-the-badge&logo=vercel)](https://trademarkcheck-sg.vercel.app/)

Search Indian trademarks fast and clearly. Run a free public trademark search by wordmark to quickly check similar records, review class coverage, and scan current application status before filing.

---

## 🚀 Features

- **Real-Time Brand Search:** Enter your proposed brand name into the search engine to instantly scan the Indian Trademark Registry.
- **Availability Checks:** Review phonetic similarities, exact matches, and class conflicts to ensure a name is safe to register.
- **Deep Case Details:** Click any result to pull up deep case logs, objection histories, and official registry document transcripts.
- **Trademark Classes Search:** An optimized search engine covering 86,715 items. Search by general class category or type a specific product (e.g., "Software", "Shampoo") to instantly find its corresponding class (Classes 1-34 for Goods, Classes 35-45 for Services).

## 🛠️ Getting Started

This is a [Next.js](https://nextjs.org/) project. Follow the instructions below to set up the project locally.

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/trademarkcheck.git
````

2. Navigate into the project directory:

```bash
cd trademarkcheck
```

3. Install dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

---

## ▶️ Running the Development Server

Start the local development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open:

```bash
http://localhost:3000
```

in your browser to view the application.
